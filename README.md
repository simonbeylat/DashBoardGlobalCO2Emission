#### Installation
You need to install these packages.
```
    - dash
    - dash_core_components 
    - dash_html_components 
    - dash_table
    - plotly
    - plotly
    - pandas 
```

Run this app with `python app.py`.
```zsh
% python3 app.py
```

visit http://127.0.0.1:8050/ in your web browser.
